📂 Portfolio System Analyst – Wahyu Septika

👤 Tentang Saya

Saya adalah seorang System Analyst dengan pengalaman dalam merancang sistem berbasis kebutuhan bisnis, menyusun dokumentasi fungsional dan teknis, serta berkolaborasi dengan tim lintas fungsi (developer, UI/UX designer dan user). Saya juga memiliki keterampilan tambahan dalam UI/UX Design, sehingga mampu menjembatani aspek bisnis, teknis dan pengalaman pengguna secara menyeluruh.

---

📌 Apa yang Akan Anda Temukan di Folder Ini

Portfolio ini berisi kumpulan dokumentasi dan desain yang menggambarkan kemampuan saya sebagai System Analyst, antara lain:
1. BRD (Business Requirement Document)
   – Menjelaskan kebutuhan bisnis dan ruang lingkup system.

2. FSD (Functional Specification Document)
   – Menjabarkan fitur, alur proses, kebutuhan fungsional sistem.

3. ERD & Database Design
   – Perancangan relasi entitas untuk mendukung implementasi teknis.

4. UI/UX Wireframe dan Design
   – Mockup antarmuka pengguna.

---

🛠️ Tools & Peran Saya dalam Setiap Dokumen

* Dokumentasi (BRD, FSD, ERD)
  → Saya menyusun dokumen ini berdasarkan analisis kebutuhan user dan proses bisnis, menggunakan tools seperti Google Docs, Draw.io, Figma dan beberapa tools pendukung.

* UI/UX Wireframe dan Design
  → Dirancang dengan Figma.
